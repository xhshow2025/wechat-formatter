# 微信公众号排版工具

一款专业的微信公众号在线排版工具，支持智能识别文章结构、多种精美模板、一键复制到微信。

## 功能特性

### 输入格式

- **纯文本** - 智能识别标题层级，自动分析文章结构
- **MD文档** - 使用Markdown语法，精确控制格式
- **DOC文档** - 标准文档格式导入

### 模板风格

12款精选模板，覆盖各种场景：


| 模板      | 风格      |
| ------- | ------- |
| 字节绿     | 科技感绿色主题 |
| 简约白     | 经典白底黑字  |
| 杂志风     | 大标题配小正文 |
| 极客黑     | 深色背景    |
| 暖时光     | 暖色生活风格  |
| 商务蓝     | 正式职场风格  |
| 文艺绿     | 小清新绿色点缀 |
| 渐变酷     | 渐变标题风格  |
| 学术灰     | 论文引用风格  |
| 苹果Bento | 苹果设计风格  |
| 蒙德里安    | 波普艺术风格  |
| 可爱粉     | 少女心彩虹风格 |


### 编辑功能

- **选中编辑** - 点击任意内容块选中
- **样式切换** - H1/H2/H3/正文/引用/列表
- **局部调整** - 字号(A-/A+)、行高(↕-/↕+)
- **增删操作** - 插入段落(⤴/⤵)、删除(×)
- **撤销重做** - Ctrl+Z 撤销

### 内容类型

支持识别和渲染：

- 一级标题(H1)、二级标题(H2)、三级标题(H3)
- 正文段落（首行缩进）
- 引用块（特殊背景和边框）
- 无序列表（圆点前缀）
- 有序列表（数字编号）
- 代码块（深色主题，支持滚动）
- 图片上传（带边框和拖拽排序）

### 其他功能

- **图片上传** - 支持多图上传，自动添加模板主色边框
- **数字高亮** - 百分比、金额等数字自动高亮
- **一键复制** - 复制HTML到剪贴板，直接粘贴到微信

## 技术栈

- **框架**: Next.js 14
- **UI**: React 18 + TypeScript
- **样式**: Tailwind CSS
- **部署**: Vercel Ready

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 启动生产服务
npm run start
```

访问 [http://localhost:3000](http://localhost:3000)

## 使用流程

1. **选择输入格式** - 点击下拉框选择纯文本/MD文档/DOC文档
2. **粘贴文章内容** - 将文章内容粘贴到左侧输入框
3. **选择模板风格** - 点击底部模板卡片切换风格
4. **调整排版** - 点击内容块选中，用工具栏调整样式
5. **复制到微信** - 点击"复制到微信"按钮，直接粘贴发布

## API配置

如需启用AI分析功能，需要配置API密钥：

1. 复制 `.env.local.example` 为 `.env.local`
2. 填写 `MINIMAX_API_KEY`
3. 重启服务

```bash
MINIMAX_API_KEY=your_api_key_here
```

## 注意事项

- AI分析功能依赖MiniMax API，需要有效的API Key
- 复制到微信建议在Chrome浏览器中操作
- 图片上传后会转为Base64格式，复制内容较大
- 部分模板在微信中可能有样式差异
